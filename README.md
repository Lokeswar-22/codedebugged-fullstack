## System Requirements

- **NODE.JS** 
- **NPM**
- **MongoDB**

## Installation

FRONTEND
```python
cd ~/path/to/your/FRONTEND
```
```python
npm install
```
```python
npm start
```
Install backend dependencies
```python
cd ~/path/to/your/BACKEND
```
```python
npm install
```
```python
npm run dev
```
